# Mika Creation Studio Software

A cinematic service Website for a creative studio offering video production, event coverage, brand development, and Software development services.

The site is built around strong motion, video-led storytelling, scroll interactions, and bold visual sections designed for a premium agency-style presentation.

## Services Highlighted

- Video production and editing
- Event coverage and media production
- Brand identity and creative direction
- Software design and development
- Campaign and launch visuals

## Tech Stack

- React
- Vite
- Tailwind CSS
- GSAP
- `@gsap/react`
- React Icons
- ESLint

## Main Features

- Scroll-driven hero video transitions
- Interactive video preview behavior
- Animated marquee intro section
- Bento-style service/project cards
- Noise texture overlays
- Responsive layout
- Production-ready Vite build setup

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:5173
```

## Available Scripts

```bash
npm run dev
```

Starts the local development server.

```bash
npm run build
```

Creates a production build in `dist`.

```bash
npm run preview
```

Previews the production build locally.

```bash
npm run lint
```

Runs ESLint checks.
